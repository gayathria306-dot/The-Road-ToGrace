# The Road to Grace

Initial project structure for a personal spiritual blog and poetry site, designed for GitHub Pages.

## Structure

- `index.html` — Homepage shell with hero and section cards.
- `pages/` — Secondary pages (`blog.html`, `poetry.html`, `about.html`).
- `templates/` — Reusable header and footer partials.
- `assets/css/` — Design system styles (`base`, `layout`, `components`).
- `assets/js/` — Shared script for injecting templates and core behavior.
- `blog/` and `poetry/` — Content folders for future entries.

## Deployment (GitHub Pages)

1. Push to your GitHub repository.
2. In repository settings, enable **Pages** from the `main` branch root.
3. Your site will publish using this static structure.
